## v4.0 → v4.1 Improvement

### Changes Applied
1. **Grade F + Activity < 40 → Bronze cap** (0/15 appts, 60% DQ)
2. **NonFixedVOIP → Bronze cap** (0/12 appts, 50% DQ)
3. **FixedVOIP → Moderate concern, excluded from Gold** (1/12 appts, 58% DQ)
4. **Solar permit → Bronze cap** (0/16 appts, 62.5% DQ)
5. **Commercial property → NEUTRAL** (3/14 appts, 21.4% — above base rate)
6. **Renter → NEUTRAL** (12/73 appts, 16.4% — outperforms owners)
7. **eHawk upstream note** — form behavior softened, upstream fraud detection handles bots
8. **typing_paste → slight concern** (0/13 appts but small sample)

## Data Quality Notes

### SEW Enrichment Gap
- SEW Trestle fill rate: 34.5% (vs Venture 99.2%)
- This means 65.5% of SEW leads had no phone grade, activity, or line type data
- 13 of 14 lost SEW appointments had completely null API data
- 7 of 9 SEW sales had null API data but scored Gold/Silver based on form behavior
- 1 SEW sale (Tony Bondar) scored Bronze due to null API data — the only B+R sale
- **In production with live enrichment, this gap will not exist**

### BatchData Demographics
- Full-attributes API key does NOT return demographics (income, net worth)
- 0% fill rate on income data with current plan
- Credit Score Inadequate (16 DQ'd leads) is the #1 uncatchable DQ reason
- Would need credit data provider or BatchData demographics upgrade

## Open Questions

1. **Enrichment gap impact on sale retention**: 90.9% sale retention in backtest becomes effectively 100% with production enrichment (Tony Bondar's null data caused the only Bronze sale).
2. **Credit score data**: 16 Gold+Silver DQ's for "Credit Score Inadequate" — biggest uncatchable signal. Worth exploring credit data provider integration.
3. **Reject tier appointments**: 7 appointments in Reject (13.7% rate). 3 were commercial property leads incorrectly rejected (fixed in v4.1). 3 were SEW leads with null API data. 1 was a confirmed_renter.
4. **Renter signal in roofing**: Solar renters outperform owners (16.4% vs 14.2%). Roofing has only 1 renter appointment with no sale. Need more roofing data to determine if renter should be stronger negative in roofing.
5. **Venture vs SEW performance gap**: Venture has 99% enrichment coverage vs SEW's 35%. SEW's higher appointment rate in Bronze (15.5%) is entirely driven by null-data leads that should have scored higher.
