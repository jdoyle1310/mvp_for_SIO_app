# v4.1 Cross-Vertical Validation Summary

**Date**: March 13, 2026
**Scorer**: Anthropic Sonnet v4 (`claude-sonnet-4-20250514`) with vertical-specific prompts
**Total leads scored**: 1,547 (1,227 solar + 320 roofing)
**Total leads with dispo**: 1,353 (1,152 solar + 201 roofing)

---

## Executive Summary

The v4.1 LLM lead scoring system has been validated across 2 verticals, 4 different buyers, and 1,353 leads with disposition data. The core finding is consistent: **filtering Bronze+Reject leads saves 24-31% of spend while retaining 85-100% of appointments and 91-100% of sales.**

| Metric | Solar (1,227 scored) | Roofing (320 scored) | Combined |
|--------|---------------------|---------------------|----------|
| Leads with dispo | 1,152 | 201 | 1,353 |
| Contacted | 869 | ~140 | ~1,009 |
| Appointments | 137 | 10 | 147 |
| Sales/Contracts | 11 (9 SEW + 2 Venture) | 1 contract | 12 |
| **Spend reduction** | **24.2%** | **31.3%** | — |
| **Appointment retention** | **85.4%** | **100%** | — |
| **Sale retention** | **90.9%*** | **100%** | — |
| **Sales in Bronze+Reject** | **1*** | **0** | **1*** |

*The 1 Bronze sale (Tony Bondar, SEW) had completely null API enrichment data — zero data from Trestle, BatchData, and TrustedForm. In production with live enrichment, this lead would score Gold/Silver. See Solar backtest doc for details.

## Validated Cross-Vertical Signals

These signals perform the same way in both solar and roofing:

### Instant Reject / Bronze Cap Signals (0% conversion in both verticals)

| Signal | Solar (n=) | Roofing (n=) | Action |
|--------|-----------|-------------|--------|
| Phone Grade F + Activity < 40 | 0/15 appts | 0% conversion | Bronze cap |
| NonFixedVOIP | 0/12 appts | Too few to validate | Bronze cap |
| Grade F + NonFixedVOIP combo | 0/5 appts | — | Reject |
| Permit on file (solar_permit / roof_permit) | 0/16 appts (solar) | 0/19 appts (roofing) | Bronze cap |
| Pre-populated form | Instant reject | Instant reject | Hard kill |
| Bot detected | Instant reject | Instant reject | Hard kill |

### Strong Positive Signals (highest conversion in both verticals)

| Signal | Solar | Roofing | Action |
|--------|-------|---------|--------|
| Grade A + Mobile + Activity 81-100 | 15.3% appt rate (n=439) | All appts/contracts in this group | Gold eligible |
| confirmed_owner + no permit | High conversion | 13% conversion, only tier with contracts | Gold |
| Grade A + confirmed_owner | 15.2% appt rate (n=368) | All 5 Gold appts had this | Gold |

### Signals That Differ Between Verticals

| Signal | Solar | Roofing | Notes |
|--------|-------|---------|-------|
| confirmed_renter | **NEUTRAL** — 16.4% appt rate, outperforms owners | **Moderate negative** — 1 appt, 0 sales | Solar renters can lease panels; roof replacement for renters is rare |
| Commercial property | **NEUTRAL** — 21.4% appt rate (above base) | **Strong negative** — residential roofers can't serve commercial | Different market dynamics |
| FixedVOIP | Moderate concern (8.3% appt rate) | Too few to validate | Keep as moderate concern both |

## Cost Analysis

### Per-Lead Scoring Cost
| Component | Cost |
|-----------|------|
| Trestle API | ~$0.01/lead |
| BatchData API | ~$0.005/lead |
| TrustedForm API | ~$0.01/lead |
| Anthropic LLM | ~$0.003/lead |
| **Total** | **~$0.03/lead** |

### ROI at Scale
At current average spend per lead:
- Solar: $63/lead average (SEW) to $58/lead (Venture)
- Filtering 24% of leads saves ~$14/lead on those leads
- Scoring costs $0.03/lead on all leads
- **Net savings: ~$3.33 per lead scored** (at 24% rejection rate)
- **111x return on scoring cost**

## What the Model Cannot Catch

These DQ reasons appear in Gold+Silver and are not detectable with the current API stack:

| Uncatchable Reason | Frequency | Potential Fix |
|--------------------|-----------|---------------|
| Credit Score Inadequate | 16 leads (solar) | Credit data provider |
| Already Has Solar | 16 leads (solar) | Partially caught by solar_permit |
| Remove From List | 16 leads (solar) | Cannot predict at scoring time |
| Incorrect Contact Info | 10 leads (solar) | Better address validation |
| Not Homeowner | 6 leads (solar) | BatchData returns null sometimes |
| Mobile/Manufactured Home | 6 leads (solar) | Hard-kill catches some |
